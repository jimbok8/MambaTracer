# CLTracer

A simple Java OpenCL Tracer mainly targeting OpenCL 1.2 with AMD and Intel Drivers. 

![Alt text](screenshot.png?raw=true "Title")

## Features

* BVH Skip Link
* Intuitive UI

## Dependencies

* JOCL Library http://www.jocl.org/

## Future Implementation

* Fast Acceleration Structure
